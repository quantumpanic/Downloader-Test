Downloader Test

Detail aplikasi:
- Saat aplikasi dimulai, akan download master file yang berisi semua link file json (ada di google drive pribadi)
- saat master file sudah selesai download, akan ada tulisan "Master file downloaded"

1. Untuk test download 
- tekan tombol "Download image"
- setelah berhasil download, akan muncul gambar hewan dan nama file

2. untuk test localization
- tekan tombol English atau Bahasa Indonesia
- setelah berhasil download language pack, akan muncul teks kata sambutan

3. untuk test database SQL
- tekan tombol "Show DB" untuk membuat database baru (dan menghapus database lama)
- tekan tombol "Add new entry" untuk menambahkan entry baru di database
- Console log akan menampilkan isi database
